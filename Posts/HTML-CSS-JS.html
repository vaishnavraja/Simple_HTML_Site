HTML, CSS and Javascript: The Big Picture

B. Styling the web with CSS

  1. CSS or Cascading style sheet is a styling machanism for the web.
  2. It is a standard that describes selectors, properties and attributes interpreted by web browsers.
  3. This standard continue to be developed as new features get added to the standard all the time.
  4. The CSS standard itself doesn't do anything. It just tells the people who created the web-browsers like Chrome, Firexbox and Edge, how to interpret the CSS language and how to style HTML with it.
  5. CSS is useful to separate style from content, where style is described in a .css documents and content in .html documents.
  6. CSS is a very rich standard that describes lots of capabilities. CSS-3 brought many advanced capabilities- like media queries, animations and gradients.
  7. There are different ways to use CSS-
    • by writing it from scratch using text editors or IDEs or
    • by using a CSS liabrary (Bootstrap) which provides pre-made styles. or
    • by writing it using CSS pre-processors (LESS) which provides a richer language to write CSS in and compile into regular CSS files, when you are done.

The web as we know it couldn't exist without CSS. It is a useful and necessary standard.
